# MyBudget - Aplikacja do zarządzania finansami

## Autor
- Ignacy Mróz

## Opis projektu
Aplikacja umożliwia śledzenie codziennych wydatków i przychodów, kategoryzowanie ich oraz wyznaczanie celów oszczędnościowych. Użytkownik może analizować swoje finanse za pomocą raportów i kontrolować limity wydatków w poszczególnych kategoriach.

## Funkcjonalności
- Pełny CRUD transakcji.
- System rejestracji i logowania.
- Import danych z plików CSV.
- Powiadomienia o przekroczeniu budżetu.
